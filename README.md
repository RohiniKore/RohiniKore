## 💼 Key Projects

### MiniSplit - Expense Splitting Application
- **Stack**: Spring Boot 3.x, React, PostgreSQL, Tailwind CSS
- **Features**: Complex split algorithms (EQUAL/PERCENTAGE/EXACT), debt simplification using greedy heap-matching, JWT authentication
- **Highlights**: Full-stack implementation with database optimization and responsive UI

### MiniLibrary - Library Management System
- **Stack**: Spring Boot, React, PostgreSQL, JPA/Hibernate
- **Features**: Book inventory management, member tracking, transaction history
- **Highlights**: Five-entity relational model with comprehensive CRUD operations

### MiniBill - Billing & Inventory System
- **Stack**: Spring Boot, React, MySQL, JWT Authentication
- **Features**: Invoice generation, inventory tracking, BigDecimal precision for financial data
- **Highlights**: Secure API with role-based access control
